► What is Cubescript?

Cubescript is a compiled scripting language built for SpongeForge mod workflows. You write .cube files that describe game logic — event handlers, world queries, item data — and the compiler catches scope errors and bad references at compile time, before the code ever touches a running Minecraft server.

Cubescript output is JavaScript loaded by a companion Sponge plugin via GraalVM's Polyglot API, running directly inside the JVM — no external Node process, no separate runtime. Mods and Sponge plugins co-exist on the same server through SpongeForge, and Cubescript slots cleanly into that pipeline.

Under the hood: Ohm parser → typed AST → static analyzer → constant-folding optimizer → JavaScript codegen. The full grammar is in src/cubescript.ohm.

.cube source → Ohm parse → AST → Analyze → Optimize → JS codegen → GraalVM / Sponge
