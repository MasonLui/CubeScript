Cubescript

Statically checked scripts for the Sponge mod ecosystem — catch errors before the server loads.

Cubescript is a compiled language designed for SpongeForge mod workflows. You write .cube files; the compiler catches undefined names, duplicate bindings, and type errors at compile time; the output is JavaScript loaded by a companion Sponge plugin via GraalVM's Polyglot API — running directly inside the JVM, no external Node process required.

The pipeline is: Ohm grammar → AST → static analysis → constant folding → JS codegen. Every compile-time rule is listed in the README. As types and control flow are added, both this page and the README will be updated. Anything not yet shipped is marked Not yet implemented.

.cube source → cubescript compile → JavaScript (ES module) → Sponge plugin (Java + GraalVM) → Minecraft server

Source on GitHub · Grammar: cubescript.ohm.

Example .cube → generated JS 

// Cubescript source (.cube)
let cobble_stacks = 1 + 2;
let greeting = "Hello, Overworld!";
greeting;

// Generated JavaScript (loaded by Sponge via GraalVM)
let cobble_stacks = 3;
let greeting = "Hello, Overworld!";
greeting;
